# 样式转换器

## 介绍
这是一个函数

实现：`短横线命名的css字符串` 和 `驼峰命名的css对象` 之间进行相关转换

## 兼容性

<SupportTable  H5  VUE2 />

## 安装

### 使用全局挂载的方法
参考:[全局安装](/README.html#全局注册)
```js
const cssConverter = this.$h.cssConverter;
```

### 按需导入
```js
import cssConverter from 'h-uni/dist/for-vue2/util/cssConverter';
```
